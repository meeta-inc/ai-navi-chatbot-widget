🔗 リンク専用チャットボットサンプル

トリガーボタンなしでリンクのみでチャットボットを使用

このページは右下のトリガーボタンなしで、リンクとボタンのみでチャットボットを開く例です。

ウェブサイトの自然な流れにチャットボットを統合したい時に便利です。

🎯 すっきりしたUI

トリガーボタンがないため、よりすっきりとしたユーザー体験を提供します。

🔗 自然な統合

既存のウェブサイトのボタンやリンクに自然にチャットボット機能を追加できます。

⚡ 簡単な設定

showTriggerButton: false オプション一つで簡単に設定できます。

💬 カスタマーサポートが必要ですか？

いつでもお気軽にお問い合わせください！専門スタッフがサポートいたします。

💬 相談開始 📞 即座にお問い合わせ ❓ ヘルプ

📝 実装方法

以下のように設定するとトリガーボタンなしでリンクのみでチャットボットを使用できます：

<script> window.chatbotWidgetOptions = { title: 'カスタマーサポート', primaryColor: '#4299e1', secondaryColor: '#3182ce', position: 'right', showTriggerButton: false // トリガーボタンを非表示 }; </script> <script src="https://embedded-script-ainavi-dev.meeta.jp/chatbot-widget.iife.js"></script> <!-- チャットボットを開くリンク --> <a href="#" data-chatbot-trigger>お問い合わせ</a> <button data-chatbot-trigger>カスタマーサポート</button> <span class="chatbot-trigger-link">ヘルプ</span>
