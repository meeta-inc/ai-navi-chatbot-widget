🤖 챗봇 위젯 테스트 페이지

이 페이지는 챗봇 위젯의 동작을 테스트하기 위한 샘플 페이지입니다.

화면 오른쪽 하단에 챗봇 트리거 버튼이 표시되며, 클릭하면 모달이 슬라이드되어 나타납니다.

📋 기본 사용 방법

다른 웹사이트에서 이 위젯을 사용하려면 다음 스크립트를 추가하세요:

<script src="https://your-domain.com/chatbot-widget.iife.js"></script>

🎨 커스터마이징 옵션

위젯의 제목, 색상, 위치 등을 커스터마이징할 수 있습니다:

<script> // 설정 옵션 정의 window.chatbotWidgetOptions = { title: '💬 고객 지원', primaryColor: '#4299e1', secondaryColor: '#3182ce', position: 'right', chatbotUrl: 'https://your-chatbot-url.com' }; </script> <script src="https://your-domain.com/chatbot-widget.iife.js"></script>

🔧 사용 가능한 옵션들:

title
모달 헤더에 표시될 제목
primaryColor
주 색상 (hex 코드)
secondaryColor
보조 색상 (hex 코드)
position
'left' 또는 'right'
chatbotUrl
챗봇 iframe URL

🧪 실시간 테스트

아래 버튼들을 클릭하여 다양한 설정으로 위젯을 테스트해보세요:

✨ 주요 개선사항

  • 컴팩트한 헤더: 패딩과 높이를 최소화하여 더 많은 콘텐츠 공간 확보
  • 설정 가능한 옵션: 제목, 색상, 위치를 클라이언트에서 쉽게 커스터마이징
  • 반응형 디자인: 모든 디바이스에서 최적화된 경험
  • 부드러운 애니메이션: 현대적이고 세련된 사용자 경험
  • 접근성 지원: 키보드 네비게이션 및 스크린 리더 지원

📜 스크롤 테스트 영역

이 영역은 스크롤 테스트를 위한 긴 콘텐츠입니다. 스크롤을 내려도 챗봇 버튼이 고정 위치에 유지되는지 확인해보세요.
