🤖 チャットボットウィジェットテストページ

このページはチャットボットウィジェットの動作をテストするためのサンプルページです。

画面右下にチャットボットトリガーボタンが表示され、クリックするとモーダルがスライドして表示されます。

💡 ヒント: トリガーボタン以外にも、下の「リンクトリガーテスト」セクションのボタンでもチャットボットを開くことができます！

📋 基本的な使用方法

他のウェブサイトでこのウィジェットを使用するには、次のスクリプトを追加してください：

<script src="https://your-domain.com/chatbot-widget.iife.js"></script>

🎨 カスタマイズオプション

ウィジェットのタイトル、色、位置などをカスタマイズできます：

<script> // 設定オプション定義 window.chatbotWidgetOptions = { title: '💬 カスタマーサポート', primaryColor: '#4299e1', secondaryColor: '#3182ce', position: 'right', chatbotUrl: 'https://your-chatbot-url.com' }; </script> <script src="https://your-domain.com/chatbot-widget.iife.js"></script>

🔧 利用可能なオプション：

title
モーダルヘッダーに表示されるタイトル
primaryColor
メインカラー（hexコード）
secondaryColor
サブカラー（hexコード）
position
'left' または 'right'
chatbotUrl
チャットボット iframe URL
